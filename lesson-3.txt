
Bài 1: Giải thích câu lệnh
a. git commit --amend: Sửa commit cuối cùng (message hoặc thêm file)
b. git commit --amend -m"<message>": Sửa message của commit cuối cùng
c. git restore --staged <file>: Bỏ file ra khỏi staging area (unstage)
d. git reset HEAD~1: Hoàn tác commit cuối cùng nhưng giữ lại thay đổi
